# 📖 Jie's Notes

> 我的个人公开知识库 — 全栈开发与 AI 技术沉淀

嘿，你好呀 👋

我是一名热爱技术的青年，喜欢折腾各种新奇的玩意儿。代码写累了就去翻翻技术博客，看到有意思的东西就忍不住想动手试试。这个小站就是我的数字花园，用来记录学习路上的所思所想、踩过的坑和偶尔的灵光一现。

写代码的时候觉得世界特别安静，解 Bug 的时候又觉得世界特别嘈杂。但不管怎样，能把自己脑子里的想法变成屏幕上跑起来的东西，这件事本身就够酷了。

如果你路过这里，刚好看到某篇文章对你有帮助，那我会很开心 😄

## 🛠️ 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建静态文件
npm run build

# 预览构建产物
npm run preview
```

## 📦 技术栈

- [VitePress](https://vitepress.dev/) — 静态文档生成器
- [Vite](https://vitejs.dev/) — 前端构建工具
- [Markdown](https://markdown.com.cn/) — 内容编写
