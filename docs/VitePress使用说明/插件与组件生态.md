## 二、一站式组件库

### `@theojs/lumen` ⭐474

专为 VitePress 打造的主题美化与 Vue 扩展组件库，一个包覆盖多种常用增强功能：

- Waline 评论系统集成
- 友情链接卡片 / 链接卡片
- 页脚定制组件
- Iconify 图标库集成
- 访问统计
- 赞助/捐赠组件

```bash
pnpm add -D @theojs/lumen
```

演示站：[lumen.theojs.cn](https://lumen.theojs.cn/)

---

### `@sugarat/theme` ⭐581

最活跃的 VitePress 博客主题，提供完整的插件全家桶：

- 离线全文搜索（pagefind）
- 图片预览
- RSS 订阅
- 评论（Artalk / Giscus）
- 返回顶部
- 全局公告

```bash
pnpm add -D @sugarat/vitepress-theme
```

## 三、Nolebase 阅读增强套件 ⭐622

[nolebase/integrations](https://github.com/nolebase/integrations) 提供一系列提升阅读体验的 VitePress 插件：

| 功能 | 说明 |
|------|------|
| **Inline Link Preview** | 内联链接预览，无需跳转即可查看目标内容（类似 Obsidian） |
| **Bi-Directional Links** | 双向链接，支持 `[[wiki]]` 语法 |
| **Highlight Targeted Heading** | 锚点跳转时目标标题产生闪烁高亮动画 |
| **Thumbnail Hashing** | 基于 ThumbHash/BlurHash 的图片缩略图，懒加载时显示模糊占位图 |
| **Enhanced Readabilities** | 页面可读性增强（字号、行距等优化） |
| **Enhanced `<mark>`** | 高亮标记元素的视觉增强 |
| **Git-based Page History** | 展示页面的 Git 变更历史 |
| **Breadcrumbs** | 面包屑导航组件 |
| **Page Meta** | 自动生成 SEO 友好的 meta 标签和 OG 预览图 |

```bash
# 按需安装
pnpm add -D @nolebase/vitepress-plugin-highlight-targeted-heading
pnpm add -D @nolebase/vitepress-plugin-inline-link-preview
# ... 其他插件
```

## 四、VitePress 原生隐藏功能

除了社区插件，VitePress 本身也有一些不太为人知的实用功能：

| 功能 | 说明 |
|------|------|
| **`` ```js-vue `` 代码块** | 在代码块中启用 Vue 模板插值，可以动态注入变量 |
| **`<style module>`** | Markdown 中支持 CSS Modules，避免样式冲突 |
| **`cleanUrls: true`** | URL 不带 `.html` 后缀，更简洁美观 |
| **`metaChunk: true`** | Meta 信息代码分割优化，减少主 JS 体积 |
| **`ignoreDeadLinks`** | 关闭死链检测，避免示例 URL 报错 |
| **`postRender` hook** | 构建后钩子，可在最终 HTML 中注入自定义内容（如 CDN 脚本） |
| **`-vue` 代码后缀** | 支持 `ts-vue`、`js-vue`、`md-vue` 等，在代码块中使用 Vue 插值 |
