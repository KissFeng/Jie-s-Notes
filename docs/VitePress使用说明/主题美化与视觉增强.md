整理自 VitePress 官方文档、社区主题和优秀案例，收录背景效果、动画、布局覆盖等视觉增强方案。

## 一、首页 Hero 粒子动画

用 `tsparticles` 替换默认的 Hero 图片区域，实现星空连线/彩色粒子飘浮效果。

### 安装

```bash
pnpm add -D @tsparticles/engine @tsparticles/vue3 @tsparticles/slim
```

### 创建组件 `ParticlesBg.vue`

```vue
<template>
  <div id="tsparticles" class="particles-container"></div>
</template>

<script setup>
import { onMounted } from 'vue'
import { loadSlim } from '@tsparticles/slim'
import { tsParticles } from '@tsparticles/engine'

onMounted(async () => {
  await loadSlim(tsParticles)
  await tsParticles.load('tsparticles', {
    particles: {
      number: { value: 60, density: { enable: true, area: 800 } },
      color: { value: '#646cff' },
      links: { enable: true, color: '#646cff', opacity: 0.3 },
      move: { enable: true, speed: 1.5 },
      size: { value: { min: 1, max: 3 } },
      opacity: { value: 0.5 }
    },
    interactivity: {
      events: {
        onHover: { enable: true, mode: 'grab' }
      },
      modes: {
        grab: { distance: 140, links: { opacity: 0.6 } }
      }
    },
    background: { color: 'transparent' }
  })
})
</script>

<style scoped>
.particles-container {
  width: 100%;
  height: 300px;
}
</style>
```

### 注册到主题

```ts
// .vitepress/theme/index.ts
import { h } from 'vue'
import DefaultTheme from 'vitepress/theme'
import ParticlesBg from './components/ParticlesBg.vue'

export default {
  extends: DefaultTheme,
  Layout() {
    return h(DefaultTheme.Layout, null, {
      'home-hero-image': () => h(ParticlesBg)
    })
  }
}
```

### 效果说明

- 星空背景：粒子之间产生连线，鼠标靠近时连线加亮
- 可调参数：粒子数量、颜色、速度、连线透明度等
- 也可用 `@tsparticles/vue3` 组件方式引入（更 Vue 风格）

## 二、暗黑模式切换圆形动画

VitePress 官方文档中介绍了一种基于浏览器 **View Transitions API** 的酷炫效果：点击暗黑模式切换时，从鼠标位置产生一个圆形裁剪展开/收缩动画。

### 原理

利用 `document.startViewTransition` + CSS `::view-transition` 伪元素，在切换时创建一个从点击位置扩展到全屏的圆形遮罩。

### 实现

```vue
<!-- .vitepress/theme/components/ToggleAppearance.vue -->
<template>
  <button @click="toggle" class="toggle-btn" title="切换主题">
    <span v-if="isDark">🌙</span>
    <span v-else>☀️</span>
  </button>
</template>

<script setup>
import { ref, onMounted } from 'vue'

const isDark = ref(false)

onMounted(() => {
  isDark.value = document.documentElement.classList.contains('dark')
})

function toggle(event) {
  // 检查浏览器是否支持 View Transitions API
  if (!document.startViewTransition) {
    applyTheme()
    return
  }

  const x = event.clientX
  const y = event.clientY
  const endRadius = Math.hypot(
    Math.max(x, innerWidth - x),
    Math.max(y, innerHeight - y)
  )

  const transition = document.startViewTransition(() => {
    applyTheme()
  })

  transition.ready.then(() => {
    const clipPath = [
      `circle(0px at ${x}px ${y}px)`,
      `circle(${endRadius}px at ${x}px ${y}px)`
    ]
    document.documentElement.animate(
      { clipPath: isDark.value ? clipPath : clipPath.reverse() },
      {
        duration: 400,
        easing: 'ease-in',
        pseudoElement: isDark.value
          ? '::view-transition-new(root)'
          : '::view-transition-old(root)'
      }
    )
  })
}

function applyTheme() {
  isDark.value = !isDark.value
  document.documentElement.classList.toggle('dark', isDark.value)
}
</script>
```

### 配套 CSS

```css
/* 降低动画层级冲突 */
::view-transition-old(root),
::view-transition-new(root) {
  animation: none;
  mix-blend-mode: normal;
}
```

### 注意

- 需要检查 `prefers-reduced-motion`，尊重用户的动画偏好设置
- 部分浏览器可能不支持该 API，需做降级处理（直接切换，无动画）

## 三、Layout 插槽系统

VitePress 默认主题提供了 **20+ 个插槽**，可以在不修改主题源码的情况下注入自定义组件。

### Doc 布局插槽（默认）

| 插槽名 | 位置 |
|--------|------|
| `doc-top` / `doc-bottom` | 文档区域顶部 / 底部 |
| `doc-before` / `doc-after` | 文档内容前后 |
| `doc-footer-before` | 文档页脚之前 |
| `sidebar-nav-before` / `sidebar-nav-after` | 侧边栏导航前后 |
| `aside-top` / `aside-bottom` | 右侧栏顶部 / 底部 |
| `aside-outline-before` / `aside-outline-after` | 大纲前后 |
| `aside-ads-before` / `aside-ads-after` | 广告位前后 |

### Home 布局插槽

| 插槽名 | 位置 |
|--------|------|
| `home-hero-before` / `home-hero-after` | Hero 区域前后 |
| `home-hero-info` | Hero 信息区域 |
| `home-hero-actions-before-actions` / `home-hero-actions-after-actions` | 按钮前后 |
| `home-hero-image` | Hero 图片区域（粒子动画就替换这里） |
| `home-features-before` / `home-features-after` | 特性区域前后 |

### 全局插槽（所有页面可用）

| 插槽名 | 位置 |
|--------|------|
| `layout-top` / `layout-bottom` | 整个布局顶部 / 底部 |
| `nav-bar-title-before` / `nav-bar-title-after` | 导航标题前后 |
| `nav-bar-content-before` / `nav-bar-content-after` | 导航内容前后 |
| `nav-screen-content-before` / `nav-screen-content-after` | 移动端导航内容前后 |

### 使用方式

**方式一：Vue 模板（推荐）**

```vue
<!-- .vitepress/theme/MyLayout.vue -->
<script setup>
import DefaultTheme from 'vitepress/theme'
import MyComponent from './components/MyComponent.vue'
</script>

<template>
  <DefaultTheme.Layout>
    <template #aside-outline-before>
      <MyComponent />
    </template>
  </DefaultTheme.Layout>
</template>
```

**方式二：渲染函数**

```ts
import { h } from 'vue'
import DefaultTheme from 'vitepress/theme'
import MyComponent from './components/MyComponent.vue'

export default {
  extends: DefaultTheme,
  Layout() {
    return h(DefaultTheme.Layout, null, {
      'aside-outline-before': () => h(MyComponent)
    })
  }
}
```

## 四、CSS 变量自定义

VitePress 默认主题的颜色、字体等都可以通过覆盖 CSS 变量来定制。

### 品牌色

```css
/* .vitepress/theme/custom.css */
:root {
  --vp-c-brand-1: #646cff;   /* 主色 */
  --vp-c-brand-2: #747bff;   /* hover */
  --vp-c-brand-3: #535bf2;   /* active */
  --vp-c-brand-soft: rgba(100, 108, 255, 0.14); /* 浅色背景 */
}

.dark {
  --vp-c-brand-1: #646cff;
  --vp-c-brand-2: #747bff;
  --vp-c-brand-3: #535bf2;
  --vp-c-brand-soft: rgba(100, 108, 255, 0.16);
}
```

### 自定义字体

```css
:root {
  --vp-font-family-base: 'Your-Font', 'Inter', sans-serif;
  --vp-font-family-mono: 'Your-Mono-Font', monospace;
}
```

如果使用自定义 `@font-face`，建议从 `vitepress/theme-without-fonts` 导入主题，避免加载默认的 Inter 字体：

```ts
// .vitepress/theme/index.ts
import Theme from 'vitepress/theme-without-fonts'
import './custom.css'
```

### 背景与卡片

```css
:root {
  --vp-home-hero-name-color: transparent;
  --vp-home-hero-name-background: linear-gradient(135deg, #646cff 0%, #a855f7 100%);
  --vp-home-hero-image-background-image: linear-gradient(135deg, #646cff50 0%, #a855f750 100%);
  --vp-home-hero-image-filter: blur(56px);
}
```

## 五、渐变网格背景（纯 CSS）

不用任何第三方库，纯 CSS 实现一个极光风格的首页背景：

```css
/* .vitepress/theme/custom.css */
.VPHero .VPImage {
  filter: blur(80px);
  opacity: 0.6;
}

/* 极光渐变背景 */
.VPHomeHero::before {
  content: '';
  position: absolute;
  inset: 0;
  background:
    radial-gradient(circle at 20% 50%, rgba(100, 108, 255, 0.15) 0%, transparent 50%),
    radial-gradient(circle at 80% 20%, rgba(168, 85, 247, 0.12) 0%, transparent 50%),
    radial-gradient(circle at 50% 80%, rgba(59, 130, 246, 0.1) 0%, transparent 50%);
  pointer-events: none;
  z-index: -1;
}
```

效果：柔和的多色渐变光晕，无 JS 依赖，性能开销极小。

## 六、内部组件覆盖

通过 Vite 的 `resolve.alias` 可以替换默认主题的内部组件，例如替换导航栏、侧边栏等。

```ts
// .vitepress/config.mts
import { fileURLToPath } from 'node:url'

export default defineConfig({
  vite: {
    resolve: {
      alias: [
        {
          find: /^.*\/VPNavBar\.vue$/,
          replacement: fileURLToPath(
            new URL('./theme/components/MyNavBar.vue', import.meta.url)
          )
        }
      ]
    }
  }
})
```

> ⚠️ 覆盖内部组件时需注意：组件名可能在小版本之间变化，建议参考 VitePress 源码确认具体路径。
