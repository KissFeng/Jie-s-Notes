# 树立正确政绩观学习教育

## 一、背景概述

近日，中共中央办公厅印发**《关于在全党开展树立和践行正确政绩观学习教育的通知》**。

- **学习对象**：以县处级以上领导班子和领导干部特别是"一把手"为重点
- **时间安排**：2026年春节假期后启动、7月底基本结束
- **指导思想**：以习近平新时代中国特色社会主义思想为指导，深入贯彻党的二十大和二十届历次全会精神，认真落实四中全会部署

## 二、总要求

**立党为公、为民造福、科学决策、真抓实干**

以一体推进**学查改**为抓手，教育引导各级党组织和党员、干部坚持实事求是、求真务实，为人民出政绩、以实干出政绩。

## 三、工作安排

学习教育坚持聚焦主题、简约务实，不分批次、不划阶段，认真落实以下工作安排：

1. **学习研讨**：深入学习习近平总书记关于树立和践行正确政绩观的重要论述
2. **查摆问题**：通过督促检查、调查研究、了解群众反映等途径，从党性上找差距、查根源、强修养
3. **整改整治**：与中央巡视整改、中央八项规定精神学习教育整改、"十五五"规划编制实施、生态环保督察整改等相结合，边查边改、立行立改
4. **建章立制**：深入查找现行制度机制中不符合正确政绩观要求的规定，该废止的废止，该修订的修订
5. **开门教育**：查摆问题听取群众意见，整改整治接受群众监督，检验成效接受群众评判

## 四、核心要点补充

### 1. 党性决定政绩观

把握党性决定政绩观的核心论断。

### 2. 根本宗旨

牢记**为民造福是最大政绩**，立足群众急难愁盼办实事，以群众获得感检验工作成效。

### 3. 发展理念

完整全面贯彻**新发展理念**，立足高质量发展，摒弃形象工程、政绩工程，不盲目铺摊子、片面追指标。

### 4. 履职理念

践行**功成不必在我、功成必定有我**，多做打基础利长远的潜绩，兼顾当下与长远。

### 5. 务实作风

坚持实事求是，杜绝数据造假、材料政绩、重部署轻落实等形式主义问题。

### 6. 纪律规矩

严守政策法规、生态、耕地、政府债务等底线，依规依法科学决策。

### 7. 对标整改

对照政绩观错位问题自查自纠，以案为鉴，立足本职务实履职。

## 五、组织要求

- 各级党委（党组）要把开展学习教育作为重要政治任务
- 党委（党组）主要负责同志要履行好**第一责任人**责任
- 结合不同层级、地区、领域、行业实际，加强分类指导
- 做好宣传引导，力戒形式主义
