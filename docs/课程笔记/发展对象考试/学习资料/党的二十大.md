# 党的二十大

## 一、大会总论

### 大会主题

高举中国特色社会主义伟大旗帜，全面贯彻习近平新时代中国特色社会主义思想，弘扬伟大建党精神，自信自强、守正创新，踔厉奋发、勇毅前行，为全面建设社会主义现代化国家、全面推进中华民族伟大复兴而团结奋斗。

### 三个务必

务必不忘初心、牢记使命；务必谦虚谨慎、艰苦奋斗；务必敢于斗争、善于斗争。

### 新时代三件大事

1. 迎来建党一百周年
2. 中国特色社会主义进入新时代
3. 打赢脱贫攻坚战、全面建成小康社会，实现第一个百年奋斗目标

### 前进五项重大原则

1. 坚持党的全面领导
2. 坚持中国特色社会主义道路
3. 坚持以人民为中心
4. 坚持深化改革开放
5. 坚持发扬斗争精神

### 两个确立、四个意识、四个自信、两个维护

- **两个确立**：确立习近平同志党中央的核心、全党的核心地位；确立习近平新时代中国特色社会主义思想的指导地位
- **四个意识**：政治意识、大局意识、核心意识、看齐意识
- **四个自信**：道路自信、理论自信、制度自信、文化自信
- **两个维护**：坚决维护党中央权威和集中统一领导

### 两个永远在路上

全面从严治党永远在路上，党的自我革命永远在路上。

## 二、理论要点

### 党的中心任务

团结带领全国各族人民全面建成社会主义现代化强国、实现第二个百年奋斗目标，以中国式现代化全面推进中华民族伟大复兴。

### 中国式现代化五大特征

1. 人口规模巨大
2. 全体人民共同富裕
3. 物质文明和精神文明相协调
4. 人与自然和谐共生
5. 走和平发展道路

### 中国式现代化本质要求

坚持党的领导，坚持中国特色社会主义，实现高质量发展，发展全过程人民民主，丰富人民精神世界，实现全体人民共同富裕，促进人与自然和谐共生，推动构建人类命运共同体，创造人类文明新形态。

### 两步走战略

1. **2020—2035年**：基本实现社会主义现代化
2. **2035—本世纪中叶**：建成富强民主文明和谐美丽的社会主义现代化强国

### 六个必须坚持（理论方法论）

人民至上、自信自立、守正创新、问题导向、系统观念、胸怀天下。

## 三、经济社会10项重点部署

| 领域 | 重点内容 |
|------|---------|
| **经济建设** | 高质量发展是首要任务，构建新发展格局，建设现代化产业体系，完善市场经济体制，全面推进乡村振兴 |
| **科教建设** | 实施科教兴国、人才强国、创新驱动发展三大战略，实现高水平科技自立自强 |
| **民主政治** | 发展全过程人民民主，健全人民当家作主制度体系，巩固和发展爱国统一战线 |
| **依法治国** | 完善中国特色社会主义法治体系，依法行政、公正司法、建设法治社会 |
| **文化建设** | 推进文化自信自强，建设社会主义文化强国，弘扬社会主义核心价值观 |
| **民生保障** | 完善分配制度、落实就业优先、健全社保体系、推进健康中国建设，扎实推进共同富裕 |
| **生态文明** | 加快发展方式绿色转型，稳妥推进碳达峰碳中和，促进人与自然和谐共生 |
| **国家安全** | 贯彻总体国家安全观，健全国家安全体系，提升公共安全与社会治理水平 |
| **国防军队** | 如期实现建军一百年奋斗目标，加快国防和军队现代化 |
| **港澳台与外交** | 坚持一国两制，坚定推进祖国完全统一；推动构建人类命运共同体 |

## 四、全面从严治党

1. 健全全面从严治党体系，以党的自我革命引领社会革命
2. 加强党的思想建设，用习近平新时代中国特色社会主义思想凝心铸魂
3. 严明政治纪律和政治规矩，持续纠治"四风"
4. 坚持不敢腐、不能腐、不想腐一体推进，打赢反腐败斗争攻坚战持久战
