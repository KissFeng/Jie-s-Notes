# Gitee 基础配置

## 注册与创建仓库

### 注册 Gitee 账号

1. 前往 [Gitee 官网](https://gitee.com/) 注册账号
2. 完善个人信息，绑定邮箱

### 创建远程仓库

1. 登录后点击右上角 **+** → **新建仓库**
2. 填写仓库名称（如 `my-project`）
3. 选择是否初始化仓库（勾选会自动生成 README）
4. 点击 **创建** 即可

## SSH 密钥配置

### 为什么需要 SSH 密钥

+ 使用 HTTPS 方式每次 push/pull 都需要输入用户名密码
+ 使用 SSH 方式只需配置一次密钥，后续免密操作

### Step 1：生成 SSH 密钥

```bash
ssh-keygen -t ed25519 -C "your-email@example.com"
```

+ 一路回车即可（默认保存到 `~/.ssh/id_ed25519`）
+ 如果想设置密码短语（passphrase），在提示时输入

> 如果系统不支持 Ed25519，可以用 RSA：
> ```bash
> ssh-keygen -t rsa -b 4096 -C "your-email@example.com"
> ```

### Step 2：复制公钥

```bash
# Mac
cat ~/.ssh/id_ed25519.pub | pbcopy

# Windows (PowerShell)
cat ~/.ssh/id_ed25519.pub | Set-Clipboard

# Linux
cat ~/.ssh/id_ed25519.pub
# 手动选中复制
```

### Step 3：添加公钥到 Gitee

1. 登录 Gitee → 点击头像 → **设置**
2. 左侧菜单找到 **SSH 公钥**
3. 将复制的公钥粘贴到 **公钥** 文本框
4. 标题随意填写（如 "我的 MacBook"）
5. 点击 **确定**

### Step 4：验证连接

```bash
ssh -T gitee.com
```

成功会显示：`Hi 用户名! You've successfully authenticated, but Gitee.com does not provide shell access.`

## 首次关联远程仓库

### 方式一：本地项目推送到 Gitee（已有本地仓库）

```bash
# 1. 进入本地项目目录
cd my-project

# 2. 初始化 Git（如果还没有的话）
git init

# 3. 添加远程仓库地址
git remote add origin git@gitee.com:你的用户名/my-project.git

# 4. 推送到远程
git add .
git commit -m "feat: 初始化项目"
git push -u origin master
```

> `git push -u origin master` 中的 `-u` 会设置上游分支，之后直接 `git push` 即可。

### 方式二：克隆 Gitee 仓库到本地（新建仓库）

```bash
git clone git@gitee.com:你的用户名/my-project.git
cd my-project
```

## HTTPS 与 SSH 的区别

| 对比项 | HTTPS | SSH |
|--------|-------|-----|
| URL 格式 | `https://gitee.com/用户名/仓库.git` | `git@gitee.com:用户名/仓库.git` |
| 认证方式 | 每次输入用户名密码（或 Token） | SSH 密钥免密 |
| 速度 | 稍慢 | 稍快 |
| 防火墙 | 一般不受限 | 可能被防火墙拦截 |
| 推荐场景 | 临时使用、不方便配置密钥 | 日常开发（推荐） |

### 切换远程 URL 方式

```bash
# 从 HTTPS 切换到 SSH
git remote set-url origin git@gitee.com:用户名/仓库.git

# 从 SSH 切换到 HTTPS
git remote set-url origin https://gitee.com/用户名/仓库.git

# 查看当前使用的 URL
git remote -v
```

## 个人信息配置

```bash
# 全局配置（所有仓库生效）
git config --global user.name "你的名字"
git config --global user.email "your-email@example.com"

# 单个仓库配置（仅当前仓库生效，优先级更高）
git config user.name "你的名字"
git config user.email "your-email@example.com"
```

## 常见问题

### Q: push 时提示 Permission denied？

+ 检查公钥是否已添加到 Gitee
+ 检查远程 URL 是否为 SSH 格式（`git@gitee.com:...`）
+ 检查 SSH 密钥权限：`chmod 600 ~/.ssh/id_ed25519`

### Q: push 时提示 Host key verification failed？

```bash
ssh-keyscan gitee.com >> ~/.ssh/known_hosts
```

### Q: 忘记了 SSH 密钥的密码短语？

只能重新生成一对新的密钥，替换 Gitee 上的旧公钥。
