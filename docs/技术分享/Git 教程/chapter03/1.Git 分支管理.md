# Git 分支管理

## 单词

+ branch
+ checkout
+ merge

## 语法

### Git 分支建立与切换

```git
git branch &lt;branch_name&gt;			//建立分支
git checkout &lt;branch_name&gt;			//切换到 branch_name 的分支

等同于
git checkout -b &lt;branch_name&gt;		//新建 并切换到 branch_name 的分支
```

### Git 已有分支查看

```git
git branch							//查看分支
git branch -v						//查看每一个分支的最后一次提交
git branch -a						//查看所有分支 ，包含 远端分支
git branch -av 						//查看所有分支 以及 最后一次提交
```

### Git分支删除

```git
git branch -d &lt;branch_name&gt;			//删除 指定分支
git branch -D &lt;branch_name&gt;			//强制删除 指定分支，删除分支并丢掉所做的修改
```

### Git 分支合并

```git
git checkout master					//切换到分支 master
git merge &lt;branch_name&gt;				//将指定分支 的修改 合并到 master 上
```
